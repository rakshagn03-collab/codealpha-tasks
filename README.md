# 🎮 1. Hangman Game

## 📖 Description
A text and GUI-based Hangman game where the player guesses a hidden word letter by letter. The game includes ASCII graphics, input validation, restart functionality, and optional sound effects.

## ✨ Features
- Random word selection
- ASCII hangman drawing
- GUI using Tkinter
- Restart option
- Input validation
- Optional sound effects

## 🛠 Technologies Used
- Python
- Tkinter
- Random module

## ▶️ Run
```bash
python hangman.py
